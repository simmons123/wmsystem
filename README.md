# # Watermarking-System
实时多媒体数字水印系统

结合现有的深度学习领域的图像/视频水印算法的神经网络模型，与热门的Vue、Spring Boot、Pytorch等框架设计并开发，旨在为用户提供实用的版权保护功能，系统主要功能点：
  1. 用户登录注册功能；
  2. 分别实现基于图像和视频作为载体的水印嵌入/提取功能；
  3. 算法鲁棒性的验证；
